We have collected dataset of the given fish which is timelined for a period of 1 week to note the deterioration of the fish gills and eyes.
We preprocessed the dataset and separated fresh and non fresh images of fish eyes and gills.
This dataset is given into the further model to be trained and for calculating the quality parameters

Convolution Neural Network
The convolutional neural network, or CNN for short, is a specialized type of neural network model designed for working with image data. It learns multi-level features and classifier in a joint fashion.
Central to the convolutional neural network is the convolutional layer that gives the network its name. This layer performs an operation called a “convolution“.
A convolution is a linear operation that involves the multiplication of a set of weights with the input, much like a traditional neural network.
